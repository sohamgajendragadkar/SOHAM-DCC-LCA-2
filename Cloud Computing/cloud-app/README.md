# Cloud Computing Basics

## Objective
Deploy a simple application on cloud and access via browser.

## Tools Used
- Node.js
- Express.js
- Render

## Steps

1. Created Node.js app using Express
2. Tested locally on localhost:3000
3. Uploaded code to GitHub
4. Deployed app using Render
5. Accessed via browser

## Features
- Displays server time and hostname
- Multiple routes (/about, /api)
- JSON API endpoint

Successfully deployed and tested a cloud-based application.
